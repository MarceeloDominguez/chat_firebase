## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/chat_firebase.git
```

Go to the project directory

```bash
  cd chat_firebase
```

Install dependencies

```bash
  npm install
```

Firebase

```bash
  add your credentials in firebase.config.ts
```

Start the server

```bash
  npm start
```
